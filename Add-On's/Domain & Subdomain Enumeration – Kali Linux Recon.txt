1) What is Subdomain Enumeration?

Subdomain enumeration is the process of identifying subdomains associated with a domain. These subdomains often host additional services (webmail, cPanel, API servers, admin panels) that expand the attack surface.

‚∏ª

2) Why use it?
	‚Ä¢	Subdomains = extra entry points (often less secure than the main site).
	‚Ä¢	Helps understand server structure & hosting providers.
	‚Ä¢	Critical for OSINT & Penetration Testing workflows.

‚∏ª

3) How does it work?
	1.	Query subdomain search APIs (e.g., Hackertarget).
	2.	Collect associated IP addresses.
	3.	Inspect HTTP headers with curl -I.
	4.	Perform WHOIS to check ownership/hosting.
	5.	Run fingerprinting (e.g., WhatWeb) to detect frameworks & versions.
